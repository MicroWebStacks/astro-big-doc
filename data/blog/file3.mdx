# SVG
## Astro Svg

this is an Svg Astro component. Unlike `<object/>` tag, this component will be replaced with the full svg in the html page.

```jsx
<Svgb data-filename="tree.svg" />
```

<Svgb data-filename="tree.svg" />

SVG Files can be used in markdown too, as long as they don't have a `data-filename`, they'll be apssed through as is.
